ProxyLocal

ProxyLocal creates a reverse-proxy to allow you to use domains with any projects you have running on localhost:PORT

ProxyLocal creates an nginx docker container that runs on port 80, so you'll need to stop any services running on port 80 on your host machine.

1) Add sites to sites.yml

Need to add your projects to the sites.yml for port to domain mappings

  • Copy sites-example.yml
  • Edit the file to reflect your domains and their ports.
    ie: 80: docker.example.com

2) Build the proxy again (and always after editing the sites.yml). Go to the site!

  • cd /commands/
  • ./proxy-up to start the proxy
  • Go to your site
  • Without the proxy running, you should be able to view your other projects at localhost:port
  • ./proxy-down to shut down the proxy
